Política de Privacidade

Este aplicativo utiliza a API oficial do WhatsApp Business exclusivamente para envio de mensagens automatizadas a clientes que já possuem relacionamento comercial com a empresa.

Não coletamos, armazenamos ou compartilhamos dados pessoais além do necessário para o envio das mensagens.

As informações utilizadas (nome, telefone, nota fiscal, valor e vencimento) são fornecidas pelo próprio cliente no contexto comercial.

Não compartilhamos dados com terceiros.
